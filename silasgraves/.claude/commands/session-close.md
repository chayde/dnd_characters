You are ending a character development session for Silas Graves, the Level 4 Necromancy Wizard.

## Your Task

1. **Review the entire conversation history** from this session to understand what was discussed
2. **Identify and summarize:**
   - Key topics discussed (spells, background, mechanics, story, etc.)
   - Important decisions made about the character
   - Changes made to character files (character-sheet.md, spells.md, build-plan.md, background-story.md)
   - Corrections or updates to stats/abilities
   - Questions that were asked but remain unanswered
   - Topics the user wants to explore in future sessions

3. **Append a new session entry** to `currentstatus.md` using this format:

```markdown
---

## Session [DATE] - [Brief Title]

### Date
[Use today's date in format: Month DD, YYYY]

### Session Summary
[2-3 paragraph summary of what was accomplished this session]

### Key Decisions Made
- [Decision 1]
- [Decision 2]
- [etc.]

### Changes to Character Files
- **character-sheet.md:** [What changed]
- **spells.md:** [What changed]
- **build-plan.md:** [What changed]
- **background-story.md:** [What changed]
- **[other files]:** [What changed]

### Outstanding Questions
1. [Question still to be answered]
2. [Question still to be answered]
3. [etc.]

### Next Steps
- [ ] [Action item or topic for next session]
- [ ] [Action item or topic for next session]
- [ ] [etc.]

### Notes
[Any additional context or important reminders for next session]
```

## Important Instructions

- Be thorough but concise - capture the essence without excessive detail
- Focus on **decisions and changes**, not just what was discussed
- Distinguish between "decided" items and "still to be determined" items
- Make next steps actionable and specific
- If no changes were made to a file, don't list it
- Use past tense for what happened, present tense for current status

After appending the entry, confirm to the user that the session has been logged and wish them well until next time.
